# F5 Distributed Cloud DDoS Mitigation — BIG-IP GRE & BGP

Interactive documentation for configuring GRE tunnels and BGP peering
from a BIG-IP HA pair to F5 Distributed Cloud DDoS scrubbing centers
in routed mode (L3/L4).

**[View the documentation site](https://robinmordasiewicz.github.io/f5xc-bigip-ddos-gre/)**

The site lets you enter your own IPs, ASNs, and BGP passwords — every
command and diagram updates automatically.

## Local development

```bash
pip install -r requirements.txt
mkdocs serve
```
